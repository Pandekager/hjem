<interfaces>
**Systemd D-Bus API (org.freedesktop.systemd1.Manager):**
- `GetUnit(name)` → object path, raises error if not found
- `StartUnit(name, mode)` → job path
- `StopUnit(name, mode)` → job path
- `GetUnitFileState(name)` → string (disabled, enabled, etc.)

**Unit proxy interface (org.freedesktop.systemd1.Unit):**
- `ActiveState` property → "active", "inactive", "activating", "deactivating", "failed"

**Nuxt 4 server routes:**
- `server/api/mc/status.get.ts` auto-registers as `GET /api/mc/status`
- `server/api/mc/start.post.ts` auto-registers as `POST /api/mc/start`
- `server/api/mc/stop.post.ts` auto-registers as `POST /api/mc/stop`
- Handler receives `event` (H3Event), returns JSON

**dbus-next API (v0.10.2):**
- `dbus.systemBus()` — create system bus connection
- `bus.getProxyObject(dest, path)` — get proxy for a D-Bus object
- `obj.getInterface(iface)` — get typed interface
- `iface[method](...args)` — call method
- Connection is cached at module level, not per-request

**Security:**
- Origin check: validate Origin header is the dashboard's own base URL
- Rate limiting: simple in-memory counter per IP (e.g., 10 req/min)
- No auth required — private LAN (Tailscale)
</interfaces>

**Origin check:**
```ts
export default defineEventHandler(async (event) => {
  const origin = getRequestHeader(event, 'origin')
  const host = getRequestHeader(event, 'host')
  // Allow same-origin requests (no origin header = same-origin)
  if (origin && !origin.includes(host || '')) {
    throw createError({ statusCode: 403, statusMessage: 'Forbidden' })
  }
})
```

**Rate limiting (simple in-memory):**
- Track request counts per IP using a Map
- Limit: 30 requests per minute per IP
- Clean up old entries
- If exceeded, throw 429
